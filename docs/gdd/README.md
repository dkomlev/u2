# Game Design Document (GDD)

Общие дизайн-документы проекта U2.

## Документы

- **[gdd_overview.md](gdd_overview.md)** — общий обзор игрового дизайна проекта U2

GDD документы содержат высокоуровневое описание игровых механик, целей проекта, целевой аудитории и общей концепции игры.

Технические детали реализации находятся в разделе [specs/](../specs/).
